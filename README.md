📑 PRECISA-SE: AI-Powered Resume Assistant
"O que te ajudaria hoje?" (What would help you today?)

The PRECISA-SE Resume Assistant is a human-centric, open-source conversational AI designed to bridge the gap between community members and local job opportunities. Developed by CONCP, it transforms the traditional "job application" into a welcoming, guided experience.
🌍 Strategic Vision: Think Global, Act Local
Our mission is to use state-of-the-art Software Engineering and Artificial Intelligence to generate immediate impact in the neighborhood. We turn intentions into opportunities by helping People create professional digital resumes without the "blank page syndrome."
🚀 Key Features
🤖 Humanized AI Guidance: A step-by-step conversational assistant that simplifies resume creation.
🛡️ Privacy by Design: Full compliance with the Brazilian General Data Protection Law (LGPD).
♿ Inclusive & Adaptive: Initial screening to identify PwD (Persons with Disabilities) and adjust conversation complexity.
📍 Hyperlocal Focus: Connects people to vacancies within their own neighborhood to improve quality of life.
🌐 Multi-language Support: Structured for global expansion with localized prompts (PT-BR, EN, JA, and more).
🏗️ Project Structure
/docs/prompts: The "Intelligence Center" containing AI guidelines for each stage of the conversation.
/docs/workflow: Strategic mapping of the value chain (Attract → Collect → Qualify → Available → Connect).
/docs/architecture: Technical documentation of the state machine and system logic.
/src: Source code (Vite, React, Tailwind CSS, Firebase).
🛠️ Tech Stack
Frontend: React + Vite
Styling: Tailwind CSS
Backend/Hosting: Firebase (Firestore, Auth, Functions)
AI Integration: Google Gemini / LLM compatible
